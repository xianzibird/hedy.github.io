规则二：圆括号的正确使用

圆括号（parentheses）在Javascript中有两种作用，一种表示调用函数，另一种表示不同的值的组合（grouping）。我们可以用空格，区分这两种不同的括号。

规则A：调用函数的时候，函数名和左括号之间没有空格；

规则B:函数名和序列参数之间没有空格；

规则C：所有其他语法元素和左括号之间都有空格；

规则三：不要省略句末的分号

规则四：不要使用with语句

规则五：不要使用“相等”（==），只使用“严格相等”（===）

规则六：不要将不同目的的语句，合并成一行

规则七：所有变量声明都放在函数的头部

规则八：所有函数都在使用前定义

规则九：避免使用全局变量

规则十：构建函数的首字母大写

规则十一：不要使用自增（++）和自减（–），使用（+=）和（-=）

规则十二：总是使用大括号表示区块
